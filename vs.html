Visual Studio Code

1) download Visual Studio Code

Download Visual Studio Code

2) select version

3) เมื่อดาวน์โหลดเสร็จแล้ว ให้เข้าไปยังที่อยู่ไฟล์ดังนี้:
» มาที่ด้านล่างซ้าย คลิก VSCodeUserSetup....exe⌵
» เลือกShow in folder/แสดงในโฟลเดอร์

4) เมื่อเข้ามาแล้ว ส่วนมากจะอยู่ที่ Downloads
ตอนนี้เราก็ได้ตัวติดตั้ง Visual Studio Code มาเรียบร้อยแล้ว ต่อไปจะเป็นการติดตั้งโปรแกรมแล้วครับ

5) คลิกขวาที่ไฟล์VSCodeUserSetup-x64-1.xx.x.exeแล้วเลือกOpen หรือจะดับเบิ้ลคลิกเข้าไปเลยก็ได้ครับ

6) เลือก ◉ I accept the agreement และคลิกNext ＞

7) คลิกNext ＞

8) คลิกNext ＞

9) ในส่วนนี้ผมขอแนะนำให้เลือก ✅ ทุกช่องเพื่อให้การติดตั้งเข้ากับโค้ดในระบบปฏิบัติการของเราอย่างสมบูรณ์
